The release of GLScene v11 for 
Embarcadero RAD Studio Delphi & C++Builder  
Extended support for third-party libraries 
BASS, FMOD, OpenAL, ODE and Newton.  
All source files are in a Source folder and 
compiled packages are saved in bpl/dcp folders 
in the working GLScene's directory.

This version of GLScene includes bug fixes,
improvements for some features to support  
Win32/Win64 platforms for OpenGL applications.
